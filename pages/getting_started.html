For Windows users new to mining

XMR-Stak-win64.zip XMR-Stak Documentation
Unzip and run the .exe for XMR-stak on your windows PC. You will need a QRL address to setup the mining software. When prompted enter monero7 as the algo. This will allow you to mine with both CPU and GPU (supporting AMD and Nvidia) on the QRL network.
See the official documentation for XMR-Stak if you run into issues.

QRL Wallet & Documentation

QRL Wallet QRL Documentation

Setup a QRL wallet and save all of the privare key information. You will use this as the deposit address and mining worker for the pool. If you need more information see the official documentation.

xmr-stak Config

Here is an example pools.txt file. Edit this replacing the wallet_address with your QRL wallet address.

"pool_list" :
[
{"pool_address" : "qrlmining.com:5555", "wallet_address" : "YOUR_ADDRESS_HERE", "rig_id" : "", "pool_password" : "x", "use_nicehash" : false, "use_tls" : false, "tls_fingerprint" : "", "pool_weight" : 1 },
],

"currency" : "monero7",
